bookstore by darek

Projekt składać się będzie z dwóch modułów, panelu administracyjnego dostępnego tylko i wyłącznie dla użytkowników z poziomem dostępu ADMIN oraz części otwartej do której dostęp będą mieli wszyscy internauci. (za wyjątkiem obszarów które będą wymagały zalogowania się z poziomem dostępu CLIENT).

Księgarnia internetowa zostanie napisana w języku Java w oparciu o bliblioteki:

  • Spring Framework w wersji 3
      w tym:
    • Spring Security
  • Hibernate w wersji 3
      w tym:
    • Hibernate Search
    • Hibernate Validator
  • Sitemesh w wersji 3
